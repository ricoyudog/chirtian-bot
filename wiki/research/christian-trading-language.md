# Christian 交易語言模式全面參考

> 建立日期：2026-05-18 | 數據範圍：`christian log/` 2025年3月至2026年5月，共約 1297 個帖文

## 目的

本文檔為 Christian 所有交易帖文的**語言模式解析參考**，用於：
- Bot 自動解析交易指令（NLP / Regex 解析）
- 理解帖文結構與格式演進
- 建立結構化的交易動作 schema

## 核心語法模板

所有交易指令遵循以下語法：

```
[時間修飾語] + [動作動詞] + [股票代碼/名稱] + [倉位百分比] + [可選：解釋/條件]
```

### 時間修飾語

| 修飾語 | 含義 |
|--------|------|
| `開盤後` | 開盤後執行 |
| `收盤前` | 收盤前執行 |
| `下午 [時間]` | 下午指定時間 |
| `今天` | 今天 |
| `今晚` | 今晚 |
| `明天` | 明天 |
| （無） | 隱含立即執行 |

### 動作動詞

| 動詞 | 含義 | 方向 |
|------|------|------|
| `買入` | 買入 | 做多 |
| `開始買入` | 新建多頭倉位 | 做多 |
| `開始做多` | 新建多頭倉位（中文） | 做多 |
| `加倉` | 加碼 | 做多 |
| `加倉做多` | 加碼做多 | 做多 |
| `繼續加倉` | 繼續加碼 | 做多 |
| `逢低加倉做多` | 逢低買入加碼 | 做多 |
| `逢低做多` | 逢低買入 | 做多 |
| `賣出` | 賣出 | 平多 |
| `賣出全部` | 清倉 | 平多 |
| `做空` | 做空 | 做空 |
| `開始做空` | 新建空頭倉位 | 做空 |
| `short` | 做空（英文） | 做空 |

| `cover short` | 平空頭倉位 | 平空 |

### 倉位格式

| 格式 | 含義 | 範例 |
|------|------|------|
| `[N]%` | 多頭百分比（正數） | `加倉smci 1%` |
| `-[N]%` | 空頭百分比（負數） | `加倉short pltr -0.5%` |
| `（-[N]%）` | 平倉（括號負數） | `賣出tsla（-1%）` |
| `全部` | 清倉 | `賣出全部etsy` |
| `各[N]%` | 多隻各 N% | `買入ionq，rgti各1%` |

### 行格式

| 格式 | 範例 |
|------|------|
| 編號列表 | `1. 加倉smci 1%`、`2. 加倉nbis 1%` |
| 破折號列表 | `- 加倉ptir 1%` |
| 同行多筆 | `加倉aaoi 1%，加倉u 2%，加倉sofi 2%`（以 `，` 分隔） |
| 分號結尾 | `賣出cgc，賣出tlry -3%；` |

## 十、帖文標題格式

### 10.1 每日 Portfolio 帖文

**時期 A（2025-03 ~ 2025-11）**：帶「多空」
```
2025年[N]月[N]日美股港股多空
2025年[N]月[N]日美股港股多空2    // 同日後續更新
2025年[N]月[N]日美股港股多空3
2025年[N]月[N]日美股港股多空和[N]月總結  // 月初帶總結
```

**時期 B（2025-12 ~ 2026-05）**：去掉「多空」
```
2026年[N]月[N]日美股港股
2026年[N]月[N]日美股港股2
2026年[N]月[N]日美股港股3
```

### 10.2 基本面/個股分析帖文

```
2025年[N]月[股票名]                   // 例：2025年3月英偉達
2025年[N]月[主題A]-vs-[主題B]         // 例：2025年4月英偉達-vs-AMD
2026年[N]月[主題] [序號]              // 例：2026年3月CPO 2、2026年4月bottleneck 8
```

### 10.3 策略/框架帖文

```
[N]月策略調整                         // 例：10月策略調整
[N]月策略調整[N]                      // 例：10月策略調整3
美股港股多空說明                       // 說明帖
[N]月小結                            // 例：9月小結
[N]月-[M]月總結                       // 例：3月-8月總結
```

### 10.4 投票帖文

```
擇時做空對沖機制投票
擇時投機機制投票
多空最大敞口投票
做空對沖淨敞口投票
做空投票
```

### 10.5 運營/元帖文

```
即時發帖推送
頻繁交易的真相
subscription問題
微信公眾號[N]月合集說明
```

| `做多可以漲100倍以上，做空最多回報100%` | 做多為主的理念 |
| `理論上每天都可以買入股票，沒有必要捨不得賣` | 賣出觀念 |
| `重倉是一種高階技能，大多數人沒有重倉的能力` | 倉位管理觀念 |
| `基本面文章長期有用，短期沒什麼效果` | 內容價值 |

## 十五、投票帖文格式

### 結構
1. 編號要點（解釋概念）
2. 嵌入投票組件（純文本中顯示 `We were unable to load this poll.`）
3. 有時附帶投票截圖

### 投票用語
- `投票大家可以達到的做多+做空（絕對值）的最大敞口`
- `投票大家可以接受的做多+做空的最小淨做空敞口`
- `做空波動率較高，而且回報可能較慢。大家根據自己情況投票`
