# Subscribe Advanced Quotes

Webull offers an OpenAPI Advanced Market Data Subscription Service that provides access to tick-by-tick trades, order book depth, order queues, time-and-sales details, and other advanced market data.

> **Caution:** Advanced quotes subscriptions purchased through the Webull mobile app or desktop platform (QT) do not apply to OpenAPI. You need a separate subscription specifically for OpenAPI usage. Only one device may access LV1/LV2 data at any one time.

## Subscribe Advanced Quotes

1. Open and log in to the Webull Technology Official Website.
   ![Step 1](https://developer.webull.hk/apis/assets/images/advanced_quotes1-e1a4163ab85fc678a516fff35c487c19.png)
2. Click **Advanced Quotes** from the avatar menu in the upper-right corner to navigate to the Advanced Quotes Center.
   ![Step 2](https://developer.webull.hk/apis/assets/images/advanced_quotes2-d81e4af7623abd1017cf208b9530a39b.png)
3. In the Advanced Quotes Center, select **OpenAPI Advanced Quotes** to view and subscribe to the corresponding data service.
   ![Step 3](https://developer.webull.hk/apis/assets/images/advanced_quotes3-126bd534886c889c01179aec8e109750.png)

After subscribing, you can start using the advanced market data endpoints through the Data API or Data Streaming API.
