# Create FX Request

**Method:** `POST`  
**Endpoint:** `/openapi/broker/funding/fx/create`  
**Host:** `broker-api.webull.hk` (production) / `broker-api.sandbox.webull.hk` (sandbox)  
**Source:** [https://developer.webull.hk/apis/docs/reference/broker-api/broker-funding-create-fx](https://developer.webull.hk/apis/docs/reference/broker-api/broker-funding-create-fx)

Broker APIFundingFXCreate FX Request
Create FX Request
Ask AI
POST
https://hk-broker-api.uat.webullbroker.com/openapi/broker/funding/fx/create

• Function description: Create currency exchange request for Omnibus master account level. Needs to provide inputs of base currency, base amount and target currency. Will use latest FX rate for currency exchange processing.
